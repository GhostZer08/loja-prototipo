# MotoPeças Pro

Site de e-commerce de peças de moto com design moderno e animações interativas.

## 📁 Estrutura

```
motopecas-shop/
├── index.html    # Página principal com todo o HTML/CSS/estrutura
├── app.js        # Lógica JavaScript (carrinho, produtos, interações)
└── README.md     # Este arquivo
```

## ✨ Funcionalidades

- **Design Moderno**: Interface escura com tema laranja premium
- **Animações**: Efeitos float, pulse, hover, scroll reveal
- **Carrinho**: Sistema completo de adicionar/remover/atualizar quantidade
- **Filtros**: Filtragem de produtos por categoria
- **Responsivo**: Adaptável a mobile e desktop
- **Wishlist**: Sistema de favoritos
- **Toast Notifications**: Feedback visual ao adicionar itens

## 🎨 Seções

1. **Hero**: Banner principal com estatísticas e CTA
2. **Categorias**: 8 categorias de peças com cards interativos
3. **Produtos**: Grid de produtos com filtros e badges
4. **Features**: 8 vantagens com ícones animados
5. **Footer**: Links e informações de contato

## 🚀 Como Visualizar

### Opção 1: Abrir diretamente
Abra o arquivo `index.html` diretamente no navegador:
```
double-click em index.html
```

### Opção 2: Servidor local (se tiver Node.js)
```bash
npx serve .
# ou
python -m http.server 3000
```

## 🛠 Tecnologias
